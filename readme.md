## What I have so far

- Character movement and 1 simple attack
- 1 automatic spell
- Damage system
- 2 different monsters and 1 pacific mob
- Enemy spawn and waves system
- Mob drops
- UI with elapsed time and gold counter
- Game over and automatic restart
- Sound effects
- Leveling/upgrade system
- Main menu
- New map
- Wipe (every 30s)
